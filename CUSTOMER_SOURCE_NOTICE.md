# Open Source Firmware Notice

This lighting controller uses a modified build of WLED.

WLED is open-source firmware for addressable LED controllers. This installation
uses a custom version based on WLED 16.0.0 with an added House Layout interface
for mapping architectural lights on a house photo.

Original WLED project:

https://github.com/wled/WLED

Source code for the modified firmware installed on this controller:

https://github.com/Thatguy-didit/wled-house-layout
